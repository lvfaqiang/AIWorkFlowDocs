### Todo 粒度规范

- 每个 Todo 是独立可验证的原子改动
- 每个 Todo 完成后：构建验证 → 勾选 checkbox → 继续下一个
- 模块全部 Todo 完成后：用户验证 → 确认 OK → 统一 git commit
- commit message 格式：`feat(模块): 实现{功能描述}` 或 `feat(模块): T-xxx Todo标题`

> **强制规则**：未经用户确认效果，不得提交 commit。build 通过 ≠ 功能正确。
