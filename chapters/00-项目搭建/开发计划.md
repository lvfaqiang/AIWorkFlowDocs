# 鸿蒙版"现在看"App 开发计划

> 目标：1:1 还原 Android 最新代码（v1.0.0.135）功能。
> 基于四层架构（products / features / capabilities / commons）。

## 一、工期总览

| 阶段 | 内容 | 工期 | 累计 |
|------|------|------|------|
| Phase 0 | 基础设施 | 3 周 | 3 周 |
| Phase 1 | 核心 MVP | 5 周 | 8 周 |
| Phase 2 | 内容消费完善 | 4 周 | 12 周 |
| Phase 3 | 社交互动 | 3 周 | 15 周 |
| Phase 4 | 商业化 | 3 周 | 18 周 |
| Phase 5 | 高级功能 | 4 周 | 22 周 |
| Phase 6 | 打磨上线 | 2 周 | 24 周 |

单人全职约 6 个月，2-3 人并行可压缩到 3-4 个月。
建议 Phase 1 结束（第 8 周）做内部 demo 评审。

```

□ capabilities/auth-provider — 三方登录
  □ 微信登录 Adapter（@tencent/wechat_open_sdk）
  □ 微博登录 Adapter（@zyl/weibo_har）

□ features/home — 首页
  □ Banner 轮播
  □ 推荐内容列表（LazyForEach + 分页）
  □ 分类入口（横向滚动）
  □ 下拉刷新 + 上拉加载

□ features/search — 搜索
  □ 搜索页（热搜词 + 搜索历史）
  □ 搜索联想
  □ 搜索结果列表（影视/用户/动态多类型）
  □ 搜索历史本地存储（RDB）

□ features/player — 播放器
  □ 影视详情页（海报/简介/评分/演员/相关推荐）
  □ 选集列表（区分免费/付费/已购标识）
  □ 播放器 UI
    □ 播放/暂停

□ capabilities/playback 完善
  □ HLS 流媒体支持
```
