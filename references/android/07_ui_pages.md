# 07 页面交互规格

## 页面导航结构

```
SplashActivity (启动页)
  └── HomeActivity (首页 - 底部Tab)
      ├── Tab 1: 首页 (HomeFragment)
      │   ├── 顶部Banner轮播
      │   ├── Tab分类 (电影/电视剧/短剧/综艺/动漫/纪录片)
      │   └── 各分类栏目列表
      ├── Tab 2: 片库 (FilmLibraryFragment)
      │   ├── 热门推荐
      │   ├── 上新列表
      │   └── 筛选搜索
      ├── Tab 3: 看看 (KanKanFragment)
      │   └── 短视频信息流(上下滑动)
      ├── Tab 4: 动态 (DynamicFragment)
      │   └── 社区动态/影评/话题
      └── Tab 5: 我的 (MineFragment)
          ├── 头像/昵称/VIP状态
          ├── 追剧列表
          ├── 播放历史
          ├── 缓存列表
          └── 设置
```

---

## 核心页面详情

### 1. 首页 (HomeActivity + HomeFragment)

**底部导航栏**: 5个Tab（首页/片库/看看/动态/我）
- 双击Tab → `TAB_DOUBLE_REFRESH` 事件 → 刷新当前列表

**首页内容（新版 HomeIndexData）**:
- **推荐Banner**: 横向轮播，点击跳转影片详情
- **Tab分类**: 电影/电视剧/短剧/综艺/动漫/纪录片
- **每个Tab**:
  - Tab专属Banner
  - 影片网格列表（竖版封面 + 名称 + 评分 + VIP标记）
- **媒体分组**: 焦点影片 + 列表

**顶部交互**:
- 沉浸式状态栏
- 下拉刷新
- 滚动时顶部图片视差效果（`MAIN_HEAD_SWITCH_STAGE`事件）

---

### 2. 影片详情 (MovieDetailsAty) — 最复杂页面

**页面结构**:
```
┌─────────────────────────────┐
│ 播放器区域 (16:9)             │
│ ┌─────────────────────────┐ │
│ │ 火山引擎播放器             │ │
│ │ 控制栏: 进度条/清晰度/     │ │
│ │ 全屏/投屏/弹幕/倍速/选集   │ │
│ └─────────────────────────┘ │
├─────────────────────────────┤
│ 影片信息区                    │
│ 标题 / 评分 / 标签            │
│ [追剧] [分享] [收藏]          │
├─────────────────────────────┤
│ 选集列表 (电视剧)             │
│ [1] [2] [3] ... [50]         │
│ 分段: [1-50] [51-100]        │
├─────────────────────────────┤
│ 简介 / 演员 / 导演            │
├─────────────────────────────┤
│ 推荐影片                      │
├─────────────────────────────┤
│ 影评列表                      │
└─────────────────────────────┘
```

**播放器控制**:
- 清晰度切换: 480P/720P/1080P/2K/4K（高清需VIP/购买）
- 弹幕开关 + 弹幕发送
- 投屏按钮（检测局域网设备）
- 倍速: 0.5x/0.75x/1.0x/1.25x/1.5x/2.0x
- 画中画
- 全屏/竖屏切换
- 手势: 左右滑=快进/退，左侧上下=亮度，右侧上下=音量

**业务逻辑**:
- 自动恢复上次播放进度
- 电视剧自动下一集
- 播放记录自动上报
- 未登录用户播放N分钟后弹登录
- 付费内容播放前检查购买状态
- 弹幕缓存到 GreenDao (DanMuBean)

---

### 3. 片库 (FilmLibraryFragment)

**功能**:
- 热门推荐（横版大卡片滚动）
- 上新列表（竖版封面网格）
- 筛选条件: 类型/题材/年代/语言/地区/流媒体
- 排序: 热度/评分/上新
- 分页加载（下拉加载更多）

**交互**:
- 筛选条件多选，实时刷新结果
- "查看更多"跳转筛选结果页

---

### 4. 看看(短视频) (KanKanFragment)

**交互模式**: 类似抖音的全屏上下滑动
- 上滑: 下一个视频
- 下滑: 上一个视频
- 长按: 倍速播放（`SHORT_VIDEO_LONG_PRESS`）
- 双击: 点赞

**侧边栏操作**:
- 头像（点击进入作者主页）
- ❤️ 点赞
- 💬 评论
- ⭐ 收藏
- ↗️ 分享

**评论**: 底部弹出评论列表，支持回复嵌套

---

### 5. 用户中心 (MineFragment)

**已登录状态**:
```
┌─────────────────────────────┐
│ 头像  昵称  VIP标识            │
│ 关注 N | 粉丝 N | 获赞 N     │
├─────────────────────────────┤
│ 我的追剧                      │
│ 播放历史                      │
│ 我的缓存                      │
│ 我的人人币                    │
│ 我的订单                      │
├─────────────────────────────┤
│ 设置                          │
│ 意见反馈                      │
│ 关于我们                      │
└─────────────────────────────┘
```

**未登录状态**: 显示登录入口

**子页面**:
- 追剧列表: 按类型筛选(全部/电影/电视剧/动漫/综艺/纪录片)
- 播放历史: 时间倒序列表, 支持删除
- 缓存管理: 显示下载任务状态/进度/删除
- 订单列表: 分页加载, 支持跳转订单详情

---

### 6. 登录页 (LoginActivity)

**内容**:
- 顶部: Logo + "登录享受更多精彩"
- 一键登录按钮（号码认证，检测运营商自动显示手机号）
- "其他方式登录" → 展开:
  - 手机号 + 验证码输入框
  - 微信登录
  - QQ登录（已注释）
- 底部: 用户协议 + 隐私政策 复选框

---

### 7. VIP 购买页 (PayActivity)

**内容**:
- 顶部: VIP特权说明
- 商品列表: 卡片式
  - 名称/价格/原价/标签(推荐)/首购名称
  - 默认选中推荐项
- 支付方式: 支付宝/微信(Radio按钮)
- "立即开通" 按钮
- 底部: 自动续费协议说明

---

### 8. 搜索页 (SearchActivity)

**内容**:
- 顶部搜索框 + 取消按钮
- 搜索历史列表（可清空）
- 热门搜索（标签云）
- 搜索结果: 影片列表(横版卡片) + 资讯列表

---

### 9. 消息页 (MessageActivity)

**消息分类**:
- 系统消息
- 互动消息(点赞/评论/回复)
- 邀请消息

**功能**: 全部标为已读 / 按类别标为已读

---

### 10. 动态/影评模块

**发布动态**: 文字 + 图片/视频 + 话题标签
**影评**: 评分(1-5星) + 文字 + 图片 + 标签

---

### 11. 设置页

| 设置项 | 说明 |
|--------|------|
| 清晰度设置 | 默认播放清晰度选择 |
| 缓存清理 | 清除图片/视频缓存 |
| 非Wi-Fi下载 | 开关 |
| 推送设置 | 跳转系统通知设置 |
| 实名认证 | 输入身份证+验证码 |
| 账号注销 | 确认后注销 |
| 换绑手机 | 输入新手机+验证码 |
| 关于我们 | 版本号/隐私/用户协议 |

---

### 12. WebView 通用页

通过 `jsBridgeLib` 实现 Native ↔ H5 的 JSBridge 通信:
- H5调用Native: 登录/分享/支付/跳转原生页面
- Native调用H5: 刷新/回调结果

**使用场景**: 云观影、活动页、用户协议、关于我们等

---

## 全局交互

| 交互 | 实现 |
|------|------|
| 侧滑返回 | FrameModule SwipeBack |
| 下拉刷新 | SmartRefreshLayout |
| 图片加载 | Glide (FrameModule的ImageLoader扩展) |
| 吐司提示 | 自定义 ErrorToastUtils |
| 全局灰色模式 | Paint.ColorMatrixColorFilter (BaseActivity中) |
| 字体大小锁定 | Resources.updateConfiguration 覆写 |
